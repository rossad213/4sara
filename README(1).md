# 4Sara

4Sara is a browser-based menstruation, symptom, and cycle tracker.

## Features

- Onboarding with name, age, last menstruation date, average cycle length, and average menstruation length
- Returns to onboarding when no menstruation entries exist
- Menstruation logging and daily check-ins
- Custom symptoms
- Calendar with menstruation, follicular, fertile/ovulation, luteal, predicted menstruation, and check-in labels
- Conversational phase descriptions
- Dynamic phase-based insights
- Dynamic food, movement, and comfort suggestions based on tracked symptoms
- Dark mode in Settings with improved contrast
- Privacy tab
- Mobile tab
- JSON backup/restore
- CSV export
- Doctor-style report
- Smooth transitions

## Deploy

Framework: Vite  
Build command: npm run build  
Output directory: dist
