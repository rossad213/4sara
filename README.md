# 4Sara

4Sara is a browser-based period tracker web app.

## Features

- First-time onboarding with name, age, last period date, average cycle length, and average period length
- Automatically returns to onboarding if no period entries exist
- Period logging
- Daily check-ins
- Custom symptoms
- Calendar with period, predicted period, fertile estimate, ovulation, and check-in labels
- Insights
- Settings
- Privacy tab
- Mobile add-to-home-screen instructions
- Dark mode in Settings
- JSON backup/restore
- CSV spreadsheet export
- Doctor-style PDF/print report
- PIN lock for this browser
- Smooth screen transitions

## Deploy

This project is ready for Vercel.

- Framework: Vite
- Build command: npm run build
- Output directory: dist
